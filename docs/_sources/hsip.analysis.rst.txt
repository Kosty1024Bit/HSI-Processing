Анализ сигнатур
===============

Данный набор функций предоставляет основные инструменты для анализа и обработки данных в контекстах кластеризации и корреляционного анализа. Эти утилиты включают операции для вычисления центроидов и медиан, создания матриц кросс-корреляции и дополнительные методы, которые помогают в понимании структуры и взаимосвязей внутри наборов данных.
