Визуализация данных
===================

Для правильного анализа промежуточных и конечных результатов, особенно при работе с большими массивами данных, удобнее всего рассматривать их в графической форме. Для этого предоставлен функционал, использующий модуль matplotlib, с готовой надстройкой, которая упрощает использование и облегчает интерпретацию входных данных.
