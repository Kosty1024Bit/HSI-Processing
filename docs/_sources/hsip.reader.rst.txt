Функции для чтения гиперспектральных данных
===========================================

Гиперспектральные данные распространяются в разных форматах. Используя эти функции, можно читать HSI в следующих форматах: SpyFile, ENVI, AVIRIS, ERDAS/Lan, Tif.

Все функции чтения, кроме чтения в формате TIF, являются обертками для функций из `библиотеки Spectral Python <https://www.spectralpython.net/>`_.
